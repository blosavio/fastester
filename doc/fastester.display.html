fastester-defaults

A hash-map residing in src/fastester_defaults.clj that supplies the default values for the following option keys:

  • :tests-directory
  • :tests-filename
  • :html-directory
  • :html-filename
  • :img-subdirectory
  • :markdown-directory
  • :markdown-filename
  • :results-url
  • :results-directory
  • :verbose?
  • :testing-thoroughness
  • :parallel?
  • :n-threads
  • :save-benchmark-fn-results?
  • :tidy-html?
  • :preamble

Override default values by associating new values into the Fastester options hash-map. See run-all-benchmarks and generate-documents.
